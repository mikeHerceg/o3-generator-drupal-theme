<div class="documentation__container">
  <div class="documentation__col">
    <h3>Usage</h3>
    <p>
      The search input element allows users to enter terms to search through an application or website. All search input elements should include the following states:
      <ul>
      <li>Hover state</li>
      <li>Focus state</li>
      <li>Disabled state</li>
      <li>Error state</li>
      </ul>
    </p>
    <h3>Labelling Expectations</h3>
    <ul>
      <li>All search input elements should be associated with a <code>label</code>.</li>
    </ul>
    <h3>Focus Expectations</h3>
    <ul>
      <li>All search input elements should have a visible keyboard focus state.</li>
      <li>All search input buttons should have a visible keyboard focus state.</li>
    </ul>
  </div>
  <div class="documentation__col">
    <h3>Keyboard Expectations</h3>
    <ul class="documentation__keys">
      <li><code>Tab</code> = Move to next focusable form element</li>
    </ul>
    <h3>Screen Reader Expectations</h3>
    <p>
      When the search input element is focused, screen readers should announce the following:
      <ul class="documentation__keys">
        <li>Search input label</li>
        <li>Input type</li>
        <li>Search submit button</li>
      </ul>
    </p>
  </div>
</div>
